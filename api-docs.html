Middleware global

ApiKeyMiddleware
Aplicado a todos los endpoints. Valida que el header Authorization coincida con la variable de entorno API_KEY.
Headers requeridos
HeaderDescripción
Authorizationstring — valor igual a API_KEY del entorno
Errores posibles
401 No se envió el header Authorization
403 El valor de Authorization no coincide con API_KEY
